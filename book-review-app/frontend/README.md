# Book Review App - Frontend

A React + Vite frontend for the Book Review App with modern hooks, React Router, and Axios.

## Setup Instructions

1. **Install Dependencies**
   \`\`\`bash
   npm install
   \`\`\`

2. **Start Development Server**
   \`\`\`bash
   npm start
   \`\`\`

   The app will open at http://localhost:3000

3. **Build for Production**
   \`\`\`bash
   npm run build
   \`\`\`

## Features

- User Authentication (Login/Signup)
- Browse Books with Filtering
- View Book Details and Reviews
- Add Reviews with Star Ratings
- Responsive Design
- Real-time API Integration

## Project Structure

- \`/src/components\` - Reusable components
- \`/src/pages\` - Page components
- \`/src/api\` - API configuration and endpoints
- \`/src/App.jsx\` - Main app component with routing

## Environment Variables

Create a \`.env\` file (optional):
\`\`\`
VITE_API_URL=http://localhost:5000/api
\`\`\`

## Technologies Used
- React 18
- Vite
- React Router v6
- Axios
- CSS3
